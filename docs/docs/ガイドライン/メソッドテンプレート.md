# メソッドのテンプレート

```bash
# ID. メソッド名

##### 目的  
- （処理手順から抽出して要約）

##### 引数  
- （想定される引数を列挙）

##### 戻り値  
- （処理後に返却するデータ）

##### 呼出先  
- （関連しそうなクラス名.メソッド名を列挙、なければTODOと記載）

##### 処理手順  
（ユーザーが提示した手順をベースにMarkdownで整理）

##### 異常系  
- （処理手順からエラー条件を抽出、なければTODO）

##### テスト観点  
- 正常系  
  - [ ] （テスト観点を処理手順から抽出）  
- 異常系  
  - [ ] （エラー条件が発生した際の期待挙動）  
- 性能系  
  - [ ] データ件数が0件/1件/大量でも処理が正しく行われる  

##### 処理図  
```
